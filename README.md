# Проект создания виджета банковских операций клиента

## В проекте реализованы функции:
* в модуле masks функции, которые маскируют номер карты или счета
* в модуле processing функции фильтрации по состоянию и сортировки по дате
* в модуле widget реализована функция изменения формата даты и маски типа и номера карты или счета
* в модуле generators реализованы: функция фильтрации транзакций по заданной валюте, 
  функция, которая генерирует описание каждой операции, и генератор номера банковских карт 

## Установка:
1. Клонируйте репозиторий:
```git clone git@github.com:elenasyurtukova/My_project.git```
2. Установите библиотеки из pyproject.toml:
```poetry install```

## Тесты:
Каждый модуль проекта протестирован с помощью фреймворка pytest.
Чтобы запустить его (все тесты), выполните команду: ```pytest```.
Покрытие кода тестами составляет 94%
